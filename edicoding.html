EDicoding

EDicoding is a mobile application that simplifies event access for Dicoding users, allowing them to explore events, mark favorites, search quickly, and receive daily event reminders.

Technologies Used: Android Studio, Kotlin, Retrofit, Room, Glide, WorkManager, RecyclerView, ViewPager

Key Features:

  • View all Dicoding events (Upcoming & Finished)
  • Bookmark favorite events
  • Realtime search functionality
  • Dark Mode for a comfortable UI experience
  • Daily event reminders using WorkManager

Development Goals:

  • Make it easier for the Dicoding community to follow events.
  • Provide a personalized experience with bookmarks and reminders.
  • Enhance usability with Dark Mode and responsive search.
