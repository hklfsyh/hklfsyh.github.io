CookingRecipe

CookingRecipe is a mobile and web-based application designed for users to explore a variety of recipes created by professional chefs.

Technologies Used: Java, Android Studio, HTML, CSS, JavaScript, Firebase

Features:

  • Mobile app for browsing various recipes
  • Web-based admin panel to manage recipe data
  • Firebase database for storing recipe information
  • Search and categorized recommendation features
