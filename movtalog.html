Movtalog

Movtalog is a cross-platform movie catalog application that provides up-to-date movie lists with ratings, descriptions, and details.

Technologies Used: Flutter, Dart, Visual Studio Code, TMDB API

Features:

  • Real-time movie data fetched from TMDB API
  • Cross-platform support: Android, iOS, Web
  • Detailed movie information: rating, release date, and synopsis
  • Favorites feature to save movies
