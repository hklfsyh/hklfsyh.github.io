DicoStory

DicoStory is an application designed to collect and present interesting stories from various sources, especially Dicoding users. Users can add their own stories with locations so they can appear on the map.

Technologies Used: Kotlin, Android Studio, Room Database

Features:

  • View and share stories from various users
  • Location-based story mapping
  • Offline access using Room Database
  • Simple and interactive UI
