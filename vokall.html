Vokall

Vokall is a web-based Point of Sale (POS) system designed for a music-related store, managing products such as headphones, TWS, speakers, and more.

Technologies Used: PHP, MySQL, JavaScript, HTML, CSS

Features:

  • Add and remove products
  • Export and import data to/from Excel
  • Transaction and inventory management
  • User authentication and role-based access
